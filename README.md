# Adventure-Works-Sales-Dashboard-Power-BI-Project-
This project analyzes **Adventure Works sales data** to monitor **revenue, profit, orders, and return rates**.   The dashboard provides interactive insights into **product performance, regional sales distribution, and business trends**, helping organizations make data-driven decisions.

## 🔹 Project Line / Headline  
Adventure Works Sales & Profit Performance Analysis Dashboard  

## 🔹 Short Description / Purpose  
This project analyzes **Adventure Works sales data** to monitor **revenue, profit, orders, and return rates**.  
The dashboard provides interactive insights into **product performance, regional sales distribution, and business trends**, helping organizations make data-driven decisions.

## 🔹 Tech Stack  
- **Power BI** → Data visualization & dashboard creation  
- **Excel / CSV Dataset** → Data source  
- **DAX (Data Analysis Expressions)** → Calculations & KPIs  

## 🔹 Data Source  
- Adventure Works dataset including:  
  - Sales transactions  
  - Product categories & subcategories  
  - Customer and regional data  
  - Returns and order details  

## 🔹 Feature Highlights  
- **Key KPIs Dashboard:**  
  - Total Revenue ($18.51M)  
  - Total Profit ($7.86M)  
  - Total Orders (22,534)  
  - Return Rate (2.14%)  

- **Sales Analysis:**  
  - Monthly Revenue, Orders, and Returns trends  
  - Revenue vs Cost vs Profit comparison  

- **Product Insights:**  
  - Top Product by Orders  
  - Top Product by Profit  
  - Product-level performance table  

- **Regional Insights:**  
  - Sales distribution by continent (North America, Europe, Pacific)  
  - Interactive global map visualization  

- **Advanced Filtering:**  
  - Year and Month slicers  
  - Region-based filtering  
  - Category and product-level drill-down  

## 🔹 Business Impact  
- Helps identify **high-performing products and regions**  
- Detects **products with high return rates** to improve quality and strategy  
- Supports **revenue growth analysis and forecasting trends**  
- Enables **better decision-making** in sales, marketing, and supply chain
